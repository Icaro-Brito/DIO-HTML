Desafio de código 1 HTML Web Developer - DIO

Este projeto se refere ao 1º Desafio de Código da formação HTML Web Developer da DIO. Foram implementados conceitos básicos do HTML explorando os conhecimentos adquiridos ao longo do 1° modulo do curso, trazendo um breve histórico sobre como surgiu a internet, como foi a expansão da mesma e principalmente explicando comandos do HTML tanto na teoria, como também na prática, um projeto simples mas de grande satisfação pessoal.
